## Механики
- **Coyote time (койот тайм)** - возможность прыгнуть после падения с платформы
- **Jump buffer (буфер прыжка)** - запоминание нажатия на прыжок перед приземлением
- **Меню паузы**
- **Меню смерти**
- **Меню победы**
- **Режим отладки** - показ хитбоксов (включается в настройках)
- 🆕 **Кастомные уровни** — вы можете добавлять и играть свои уровни через `.txt` и `.json` файлы
- 🆕 **Автосохранение настроек** — язык, полноэкранный режим и режим отладки сохраняются и загружаются автоматически при перезапуске игры
- 🆕 **Счётчик FPS** — показывает частоту кадров (включается в режиме отладки)

### Как добавить свой уровень?

Добавить кастомный уровень очень просто! Вот как это сделать:

- Полный список поддержимаемых символов можно найти на [GitHub странице](https://github.com/SIBDuck/JumpOverlord-play/blob/main/objects/supported%20objects.md).

#### **Способ 1: Использование `.txt` файла**

- Просто напишите карту уровня в текстовом файле, используя символы, поддерживаемые игрой.

#### **Способ 2: Использование `.json` файла**
- Создайте JSON файл с ключом `"map"`. Значение должно быть списком строк, где каждая строка представляет собой один ряд вашего уровня.

Как только ваш файл будет готов, просто загрузите его в игре — и всё готово!

## Управление
- A/D - передвижение
- SPACE - прыжок
- ESC - пауза
- F1 - переключение полноэкранного режима

## Установка
1. Скачайте файл `JumpOverlord.zip`
2. Распакуйте его
3. Откройте `JumpOverlord/`
4. Запустите `Main.exe` (Python и сторонние библиотеки устанавливать не нужно)

## Системные требования

### Минимальные:
- **ОС:** Windows 10 / 11 (64-bit)
- **Процессор:** 1.5 ГГц
- **ОЗУ:** 256 МБ
- **Видеокарта:** Совместимая с DirectX 9 или OpenGL 2.0
- **Место на диске:** 200 МБ
- **Экран:** 640x360 (масштабируется до любого разрешения)

### Рекомендуемые:
- **ОС:** Windows 10 / 11 (64-bit)
- **Процессор:** 2.0 ГГц (двухъядерный)
- **ОЗУ:** 512 МБ+
- **Видеокарта:** Совместимая с OpenGL 3.0
- **Место на диске:** 200 МБ+
- **Экран:** 1920x1080 (поддерживается полноэкранный режим)
